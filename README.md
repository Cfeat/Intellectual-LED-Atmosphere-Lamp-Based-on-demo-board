# Intellectual-LED-Atmosphere-Lamp-Based-on-demo-board
基于小熊座KE1开发板的智能家居多功能氛围灯demo

本项目实现了一个基于KE1的多功能氛围灯，其应用功能丰富且交互直观。设备开机后首先显示融合了节日元素的创意开机画面：左侧显示"HELLO WORLD"问候语，右侧呈现一棵圣诞树图案，呼应即将到来的圣诞节，营造温馨氛围。

系统采用双按键交互设计，通过K1和K2实现全部功能操作。K1键用于循环切换四个功能页面：第一个页面展示实时传感器信息，包括温度、湿度、光照和声音数据；第二个页面显示睡眠环境综合评分和电池电压等信息；第三个页面为LED控制界面，清晰显示当前灯光颜色、模式（常亮/闪烁）及状态；第四个页面为智能闹钟界面，动态展示闹钟状态（空闲/倒数/响铃）。

K2键用于触发当前页面的特色功能：在传感器页面，按下K2会根据各项环境参数生成个性化的睡眠改善提示，如"温度偏低，建议调整到18-26℃"；在LED控制页面，K2可循环切换多种灯光模式，包括单色常亮、闪烁效果；在闹钟页面，K2可启动/停止闹钟，系统内置原创的渐进式闹钟铃声，采用柔和的旋律实现自然唤醒。所有操作均通过OLED显示屏提供清晰的视觉反馈，打造完整的用户体验。

外接HW483按钮传感器作为LED的总控开关。
